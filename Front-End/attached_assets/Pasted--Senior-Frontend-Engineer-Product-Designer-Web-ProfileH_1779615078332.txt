أنت Senior Frontend Engineer وProduct Designer. أريد بناء تطبيق Web احترافي اسمه مؤقتًا ProfileHub، وهو منصة مجانية لإنشاء صفحة هوية رقمية احترافية تشبه Linktree لكن أقوى: تجمع الروابط، المشاريع، الصور، الخدمات، السيرة، وسائل التواصل، QR، وتحليلات بسيطة.

المطلوب:
بناء Frontend كامل باستخدام:
- Next.js App Router
- TypeScript
- Tailwind CSS
- shadcn/ui
- React Hook Form
- Zod
- Zustand أو Context عند الحاجة
- تصميم Responsive ممتاز
- دعم RTL/LTR
- دعم العربية والإنجليزية
- Dark/Light mode
- بدون ألوان Tailwind الافتراضية المكررة
- استخدم Design Tokens من ملف مخصص
- اجعل التصميم Premium، هادئ، نظيف، وليس شبيهًا بقوالب الوكلاء الافتراضية

مهم جدًا:
سأرفع لك لاحقًا مجلد أو ملفات تصميم من مواقع تصميم خارجية. عند توفر التصميم:
- استخرج منه نظام الألوان
- استخرج spacing, radius, typography, shadows
- لا تنسخه عشوائيًا
- حوّله إلى Design System داخل Tailwind config وCSS variables
- اربط كل الصفحات بنفس الهوية البصرية

Quality:
- اكتب types واضحة
- لا تستخدم any إلا للضرورة
- افصل business logic عن UI
- اجعل الكود قابلًا للتوسع
- اجعل كل صفحة production-ready
- لا تترك TODO مبهمة
- أضف README يشرح التشغيل والبنية

Design constraints:
- لا تستخدم الأزرق/البنفسجي الافتراضي المنتشر إلا إذا كان ضمن design tokens
- لا تستخدم gradient عام ومكرر
- لا تستخدم landing page تقليدية رخيصة
- اجعل الواجهة مناسبة لمنتج SaaS عالمي
- اهتم جدًا بالموبايل لأن أغلب الاستخدام سيكون من الهاتف
